# HEALTH HEALTHER - 헬스했어? 헬스헬서!
<img src="https://user-images.githubusercontent.com/40460377/216900943-086e5781-d700-4057-bd4e-25d9718c7fcc.png"  width="320" height="100">

## 배포
https://health-healther-fe.vercel.app/

### 소개

홈짐(home gym)을 소유하고 있는 사람들이 홈짐을 개인 운동 공간 뿐만 아니라 운동을 배우고 싶은 사람들에게 알려주는 PT 공간으로 사용하여 운동도 하고 이익창출까지 할 수 있는 서비스를 제공하는 **`홈짐 예약 플렛폼`** 입니다.

### 기술 스택
 - JavaScript
 - TypeScript
 - HTML
 - tailwind CSS
 - React
 - Recoil
 
### 프로젝트 주요 기능
  1. **로그인 기능**
  
    - Oauth2와 Jwt Token을 활용한 소셜 로그인 서비스를 제공합니다.
      - 구글, 카카오
    - 최초 로그인시 회원가입 창으로 이동하여 회원정보를 입력 받습니다.
    - 회원가입이 정상적으로 완료되면 메인화면으로 이동합니다.
    
<img src="https://user-images.githubusercontent.com/40460377/216903459-7fbb6744-becb-4411-bdc7-7a1fbc1b6529.png" width="320" height="400">
  2. **게시판 기능**
  
    - 게시판 등록, 수정, 삭제 및 조회 기능을 제공합니다.
    - 회원은 게시판 타이틀을 통해 해당 키워드 가 들어간 게시판을 조회할 수 있습니다.
    - 회원은 게시판에 대해 댓글을 등록, 조회, 수정 및 삭제를 할 수 있습니다.
    - 회원이 게시판에 대해 추천할 수 있으며 원치 않을시 추천을 취소할 수 있습니다.
 
<img src="https://user-images.githubusercontent.com/40460377/216903989-36ad01f2-d7c5-453a-b3a3-89bcb1cee1b2.png"  width="320" height="400" > <img src="https://user-images.githubusercontent.com/40460377/216904140-2b36c00d-2588-43fc-97f7-1986610f9eb7.png" width="320" height="400">
    
  3. **홈짐 등록 기능**
  
    - 홈짐에 대한 등록 기능을 제공합니다.
    - 홈짐 타입별로 조회가 가능합니다.

<img src="https://user-images.githubusercontent.com/40460377/216905392-a14794d6-4621-422e-9063-b06ada46abbc.png" width="320" height="400">

  4. **쿠폰 등록 기능**
  
    - 홈짐 상품에 대한 할인 쿠폰 기능을 제공합니다.
    - 쿠폰 할인 금액과 발급 개수는 판매자 재량으로 결정할 수 있습니다.

    - 이용을 원하는 홈짐에 대한 예약 기능을 제공합니다.

  6. **후기 기능**
